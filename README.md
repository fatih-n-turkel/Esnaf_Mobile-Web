# Esnaf Web (MVP Demo)

Bu zip içindeki proje, **Next.js (App Router) + TypeScript + Tailwind + React Query + Zustand** ile hazırlanmış çalışır MVP iskeletidir.

## Kurulum
```bash
npm install
npm run dev
```

## Sayfalar
- /dashboard
- /sales/quick  (Hızlı Satış)
- /products
- /stock (demo)
- /notifications (demo)
- /settings (demo)

## Not
Bu sürüm **in-memory mock DB** kullanır (app/api + lib/mock-db.ts).
Gerçek DB’ye geçince route’ları backend’e bağlayabilirsiniz.
